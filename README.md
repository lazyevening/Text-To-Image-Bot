# Text-To-Image-Bot

## Описание
Бот умеет прикреплять текст на изображение.

## Команды
* `/start` - вернуться в стартовое меню;
* `/creators`- показать создателей бота;
* `/add_text` - при последующей отправке изображения и текста, бот совместит их.
